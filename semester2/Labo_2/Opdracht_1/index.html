De return value van de confirm() functie:

Klikt de gebruiker op "OK" dan geeft het true terug

Klikt de gebruiker op "Cancel" dan geeft het false terug

Wat is de return value als de gebruiker een tekst intypt en op 'OK' klikt?

De return value zal de ingevoerde tekst zijn die wordt weergegeven als een string.

Wat is de return value als de gebruiker op 'Cancel' klikt?

null
